
# 🚀 PulseGuard AI

AI-powered bug tracking and system efficiency optimization platform.

## Features
- Smart Bug Tracking
- AI Insights
- Automation Center
- Performance Analytics
- Power BI & Tableau Integration
- Excel Data Cleaning Support

## Frontend
- React + Vite
- Tailwind CSS
- Responsive Enterprise Dashboard

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```

Open:
http://localhost:5173
